# 阿里云ECS服务器登录指南

## 🔐 如何获取管理员账号密码

### 方法1：在阿里云控制台重置密码（推荐）

如果你忘记了密码或需要设置新密码：

1. **登录阿里云控制台**
   - 访问：https://ecs.console.aliyun.com/
   - 使用你的阿里云账号登录

2. **找到你的ECS实例**
   - 在左侧菜单点击"实例与镜像" → "实例"
   - 找到你的服务器实例（IP: 101.37.89.207）

3. **重置密码**
   - 点击实例右侧的"更多"按钮
   - 选择"密码/密钥" → "重置实例密码"
   - 输入新密码（8-30个字符，包含大小写字母、数字和特殊字符）
   - 点击"提交"

4. **重启服务器（重要！）**
   - 重置密码后，必须重启服务器才能生效
   - 点击"更多" → "实例状态" → "重启"
   - 选择"强制重启"或"正常重启"
   - 等待1-2分钟服务器重启完成

5. **使用新密码登录**
   - 使用远程桌面连接（RDP）
   - 用户名通常是：`Administrator`（Windows Server）
   - 密码：刚才设置的新密码

---

### 方法2：查看创建实例时设置的密码

如果你在创建ECS实例时设置了密码：

1. **检查创建记录**
   - 查看创建实例时的邮件通知
   - 查看阿里云控制台的"消息中心"

2. **查看实例详情**
   - 进入ECS控制台 → 实例详情
   - 查看"实例信息"中的备注或标签

---

### 方法3：使用密钥对登录（Linux系统）

如果你的服务器是Linux系统，可以使用SSH密钥对：

1. **在阿里云控制台绑定密钥对**
   - 实例详情 → "密钥对" → "绑定密钥对"

2. **使用SSH连接**
   ```bash
   ssh -i <密钥文件路径> root@101.37.89.207
   ```

---

## 🖥️ Windows Server 远程桌面连接步骤

### 第1步：获取连接信息

- **服务器IP**: 101.37.89.207
- **用户名**: `Administrator`（Windows Server默认）
- **密码**: 在阿里云控制台重置的密码

### 第2步：使用远程桌面连接

**Windows系统：**

1. 按 `Win + R` 键
2. 输入 `mstsc` 并回车
3. 在"计算机"框中输入：`101.37.89.207`
4. 点击"连接"
5. 输入用户名：`Administrator`
6. 输入密码（在阿里云控制台重置的密码）
7. 点击"确定"

**或者使用PowerShell：**
```powershell
mstsc /v:101.37.89.207
```

### 第3步：首次连接提示

- 可能会提示"无法验证此远程计算机的身份"
- 勾选"不再询问我是否连接到此计算机"
- 点击"是"继续

## 🔑 常见用户名

### Windows Server
- **默认用户名**: `Administrator`
- **其他可能**: 创建实例时自定义的用户名

### Linux系统
- **默认用户名**: `root`
- **Ubuntu系统**: `root` 或创建实例时设置的用户名

---

## ⚠️ 重要提示

1. **密码要求**（Windows Server）：
   - 长度：8-30个字符
   - 必须包含：大小写字母、数字、特殊字符
   - 不能包含用户名

2. **重置密码后必须重启**：
   - 密码修改后不会立即生效
   - 必须在阿里云控制台重启服务器
   - 等待1-2分钟重启完成后再连接

3. **安全建议**：
   - 使用强密码
   - 定期更换密码
   - 不要将密码分享给他人

## 🆘 如果无法登录

### 问题1：忘记密码
- **解决方法**：按照"方法1"在阿里云控制台重置密码

### 问题2：密码重置后仍无法登录
- 检查服务器是否已重启
- 确认用户名是否正确（Windows通常是`Administrator`）
- 检查安全组是否开放了RDP端口（3389）

### 问题3：连接超时
- 检查服务器是否运行中
- 检查安全组规则（端口3389是否开放）
- 检查本地网络连接

### 问题4：提示"此计算机无法连接到远程计算机"
- 检查服务器公网IP是否正确
- 确认服务器状态为"运行中"
- 检查Windows防火墙设置（在服务器上）
